# Sistema de Cadastro de Pessoas em C

Este repositório contém um programa em C que implementa um CRUD (Create, Read, Update, Delete)
para gerenciar um cadastro de pessoas usando arquivo binário.

## Funcionalidades
- Cadastrar nova pessoa (com verificação de CPF duplicado)
- Listar todas as pessoas cadastradas
- Buscar pessoa pelo CPF
- Atualizar os dados de uma pessoa
- Remover pessoa usando o CPF como chave única
